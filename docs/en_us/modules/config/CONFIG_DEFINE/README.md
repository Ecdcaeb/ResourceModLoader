### rml:CONFIG_DEFINE

Default at `/conifg/define`

use cfg to define a cfg.
name same as cfg. `example.cfg` -> `example.cfg`

For example:
```editorconfig
# Configuration file

##########################################################################################################
# entityelectricshakingconf
#--------------------------------------------------------------------------------------------------------#
# ap.configMisc.desc
##########################################################################################################

entityelectricshakingconf {
# ap.config.maxCountIs1.desc
B:ap_maxIs1_desc=true
}
```
